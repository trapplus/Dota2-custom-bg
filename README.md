# Как установить ?

## 1. Первый метод (готовый фон)

1. [**Скачать**](https://github.com/trapplus/Dota2-custom-bg/archive/refs/heads/main.zip) все нужные файлы.
2. Открыть папку игры:

   ```
   steam\steamapps\common\dota 2 beta\game\
   ```
3. Распаковать архив в любое удобное место.
4. Переместить папку `dota_russian` в папку игры **с заменой файлов**.
5. В параметрах запуска Dota 2 указать:

   ```
   -language russian
   ```
6. Скачать файл локализации — [Ru-locale](https://sharemods.com/ucwc15koi9ll/russian_vo.zip.html).
7. Файл с названием **`pak01_dir`** поместить в папку `dota_russian`.
8. В папку `dota_russian` **поместить один из файлов `pak03_dir`** из папки `complited-bg`.
9. **Запустить игру**.

---

## 2. Второй метод (свой фон)

1. [**Скачать**](https://github.com/trapplus/Dota2-custom-bg/archive/refs/heads/main.zip) все нужные файлы.
2. Открыть папку игры:

   ```
   steam\steamapps\common\dota 2 beta\game\
   ```
3. Скопировать папку `dota_russian` из архива **с заменой файлов**.
4. В параметрах запуска Dota 2 указать:

   ```
   -language russian
   ```
5. Скачать файл локализации — [Ru-locale](https://sharemods.com/ucwc15koi9ll/russian_vo.zip.html).
6. Файл с названием **`pak01_dir`** (размер около 500 МБ) поместить в папку `dota_russian`.
7. Видео в формате `.webm` переименовать в:

   ```
   123.webm
   ```
8. Поместить файл в:

   ```
   pak01_dir/123/
   ```
9. Запустить файл:

   ```
   create-vpk-windows.bat
   ```
10. **Запустить игру**.

## DEV-Info

### Назначение VPK пакетов
 
```
pak01_dir.vpk - RU локаль
pak02_dir     - FIX
pak03_dir+    - моды
```

* Каждый следующий мод → индекс +1.
* Конфликт индексов = мод не работает.


### Дополнительная информация

* Старые VPK из других гайдов часто не работают.
* После патчей проверяйте обновления.
* Для правок использовать **Source 2 Viewer**, не Dota 2 Tools.
