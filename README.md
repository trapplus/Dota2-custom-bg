# Кастомный фон Dota 2 

## COMPLETED-BG

Готовые фоны из Steam.
Совместимы с событиями **7.40b**.

---

## ГОТОВЫЙ ФОН (РУССКАЯ ЛОКАЛЬ)

Работает **ТОЛЬКО через `dota_russian`**.

### Установка

1. Скачать этот репозиторий.
2. Открыть:

   ```
   steam/steamapps/common/dota 2 beta/game/
   ```
3. Скопировать папку `dota_russian` из архива (с заменой файлов).
4. В параметрах запуска Dota 2 указать:

   ```
   -language russian
   ```
5. Скачать вот этот файл - [link](https://sharemods.com/ucwc15koi9ll/russian_vo.zip.html).
6. Файл с названием pak01_dir(Размер около 500 Мб) поместить в папку dota_russian. 
7. В папку `dota_russian` поместить один из файлов pak01_dir (Предварительно переименовав его в pak03_dir) в папкe `dota_russian`
8. Запустить игру.

> FIX уже встроен в `dota_russian`, отдельно ставить **не нужно**.

---

## СВОЙ ФОН (КАСТОМНАЯ ЛОКАЛЬ `dota_123`)

`dota_russian` **не трогается**.

### Сборка

1. Видео в формате `.webm` → переименовать в `123.webm`.
2. Поместить файл в:

   ```
   pak01_dir/123/
   ```
3. Запустить:

   ```
   create create-vpk-windows.bat
   ```
4. Переименовать `pak01_dir.vpk` → `pak03_dir.vpk` (Если такой уже сущестует, удалить его и скопировать наш только что созданный).
5. Скопировать в: (если отсутствует, создать, добавить все файлы из dota_russian)

   ```
   steam/steamapps/common/dota 2 beta/game/dota_123/
   ```
6. В параметрах запуска указать:

   ```
   -language 123
   ```

## ИНДЕКСЫ VPK (ВАЖНО)

```
pak01_dir.vpk — RU локаль
pak02_dir     — FIX
pak03_dir+    — моды
```

* Каждый следующий мод → индекс +1.
* Конфликт индексов = мод не работает.

---

## ВАЖНО

* Старые VPK из других гайдов часто не работают.
* После патчей проверяйте обновления.
* Для правок использовать **Source 2 Viewer**, не Dota 2 Tools.
